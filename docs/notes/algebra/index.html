Modular Arithmetic

  • Defn: If a and b are integers, a m is a positive integer, then a is congruent to modulo b, if, m divides a - b (i.e. the delta is divisible by m)
  • The above holds only if a % m == b % m
  • Also, congruence holds true if and only iff, there exists a number k, such that a = b + km, (writing the first point in another way. If m id divisible, then there must be a number k)
  • Congruence Class: A set of all numbers congruent to a particular number

Properties

  • If a == b mod m and c == d mod m, then
    • a + c == b + d mod m
    • ac = bd mod m
    • Corollary (a + b) mod m == ((a mod m) + (b mod m)) mod m
    • Corollary ab mod m == ((a mod m) * (b mod m)) mod m

Applications

  • Cryptography
  • Hashing
  • Generating pseudo random numbers

Modular Exponentiation

  • Solves the problem of b^n mod m
  • Intuition: Use binary expansion of n to compute power of any number.
  • e.g. 3^11 = 3^8 * 3^2 * 3^1
  • When the number/power get larger, take the mod

Modular Multiplicative Inverse

  • Modular Mulplicative Inverse a of an integer x is congruent to 1 modular some modulus m a.x == 1 mod m

Ref: Rosen, Kenneth H. Discrete Mathematics and it’s Applications. Section 3.4
