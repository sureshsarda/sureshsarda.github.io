Hashing

  • Polynomial Rolling Hashing
hash(s) = s[0] + s[1]**p + s[2]**p^2 ... s[n-2].p^(n-2) + s[n-1].p^(n-1) mod m
        = sum s[i].p^i mod m
    
where p and m are some chosen numbers 

p <- 31
m <- 1e9 + 9
hash_value <- 0
p_pow <- 1

for char_as_number in string:
    hash_value <- (hash_value + char_as_number ^ p_pow) % m
    p_pow <- (p_pow * p) % m

At the end of the loop you will have the hash value.
