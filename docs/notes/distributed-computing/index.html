Leader Election

Bully Algorithm

Message Types

  • Election
  • Victory
  • Coordinator

Algorithm

  • If highest processID - Send Victory Message, Else, broadcast Election to higher processes
  • If no answer after sending election, send Victory to all processes
  • If answer received, then further messages
  • If receives Election message - ack and go to top
  • If coordinator message - treat sender as coordinator
graph TD
    A{Highest ID} -- Yes --> B[Send Victory message]
    A -- No --> C[Send Election Message]
    B --> D
    C -- Ack Received --> D((Leader Elected))
    C -- No Answer --> E[Send Victory]
    E --> D
    
    T((Election Triggered)) --> X
    X[Election Message] --> Y[Send Ack]
    Y --> A
