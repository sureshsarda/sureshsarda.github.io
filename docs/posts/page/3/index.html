Weekly Unix #2 - du

du = Disk Usage du summarizes the disk usage for files and folders. How to use du This will list size of all the files as well as folders in the current directory. The folders will be explored recursively. That means all the files in the folders will be listed as well. The above gets bit out of hand when you have lot of files. Practical Emacs Tip #1: Meet the C-x z command

The Emacs manual simply states that “Repear the most recently executed command”. This command just repeats the last command with the last given set of arguments to that command (if any). No rocket science. Just simple Emacs! By the way, before we start there is one more thing - Once you press C-x z, you can repeatedly press only z for the command to repeat itself! No need to press C-x again and again.
